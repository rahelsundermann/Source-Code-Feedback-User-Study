# Documents of the User Study

## Source Code
In the folder "Source Code" you can find the source code of both tasks. This is the code containing coding conventions which was given to the students to be improved.

## User Feedback
In the folder "User Feedback" are the results of the questionnaire which the students had to answer after completingg both tasks. 
Please note, that the User Feedback also contains the answer of those 7 students whose results were removed from our evaluation.
